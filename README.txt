DirectMusicPlatform (DMP)

Wie erstelle und verroute ich eine page:
    1. erstelle eine Page (in Django auch App genannt) mit dem Befehl "python3 manage.py startapp nameOfPage" im Terminal
    2. füge die App im settings.py file in den Apps hinzu
    3. views.py in der App erstellen
    4. views hinzufügen
    5. verrouten im main urls.py und im urls.py der App
